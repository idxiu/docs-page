# Git工具

如果您的电脑已安装Git命令行工具，请跳过本章。

# MacOSX

MacOSX 部分机型自带git工具，如果没有，可以通过brew (opens new window)安装git命令工具

安装命令：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

brew install git

# 问题排查

打开操作系统终端命令行软件，输入git --version, 检查下命令是否存在。

如果终端出现下列错误，则说明git没有安装成功。

  • 错误1： 'git' 不是内部或外部命令，也不是可运行的程序或批处理文件。
  • 错误2：git: command not found

特别注意：windows电脑，如果您确定已成功安装Git安装包，但是命令行还是出现上面的错误，请将git安装目录加入到系统环境变量。
