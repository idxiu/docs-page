    # Attributes

    参数 说明 类型 可选值 默认值
    openType 微信开放能力 string getPhoneNumber
    getuserinfo
    opensetting    		 null
    disabled 是否禁用 Boolean — false
    title 按钮文字 string — —
    bgColor 背景色 string — 0
    width 宽度 string/number — 710
    height 高度 string/number — 88
    borderRadius 圆角 string/number — 10
    isBottom 是否固定在底部 Boolean — false
    waring 是否警告 Boolean — false

    # Event

    事件 说明
    click 点击事件
    getPhoneNumber 获取手机号码
    getuserinfo 获取用户信息
    opensetting 打开授权设置页
