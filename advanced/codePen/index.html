# 代码块

一直以来我都有做笔记的习惯，每次有学到新的东西，我都会记在自己的有语雀笔记上，久而久之就记录了很多比较小的代码块/知识点。

这类代码块/知识点，因为比较小，一段话，一段代码就能说清楚, 很难整理成一个博客,这里就是用来记录这些东西的。

这类代码块也通常比较偏，所以在今后积累的比较多了，这也是很有价值的内容，可以快速学习到很多内容！
